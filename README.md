# OmniKit Issue Tracker

Welcome to the official public issue tracker for **OmniKit** - The Ultimate Developer Toolkit.

Since the main codebase for OmniKit is private, we use this repository to transparently track known issues, collect user feedback, and collaborate with our community.

## 🐛 Found a Bug?
If you encounter a bug or something isn't working as expected in any of our tools, please let us know by [Opening a Bug Report](https://github.com/omnikit-tools/omnikit-issues/issues/new?template=1_bug_report.yml).

## 💡 Have an Idea?
Do you have a cool idea for a new developer tool? Or want to enhance an existing one? We'd love to hear it! [Submit a Feature Request](https://github.com/omnikit-tools/omnikit-issues/issues/new?template=2_feature_request.yml).

## 🔒 Security Vulnerabilities
If you discover a security vulnerability, please do NOT open a public issue. Instead, refer to our [Security Policy](./SECURITY.md) for instructions on how to report it responsibly.

*Thank you for helping us make OmniKit better!*
