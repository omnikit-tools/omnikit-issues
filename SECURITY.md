# Security Policy

## Supported Versions

Currently, **OmniKit** is actively maintained and we take security very seriously. 

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub issues.**

If you believe you have found a security vulnerability in OmniKit (e.g., exposing sensitive environment variables, XSS in our tools), please report it to us privately via email at: **[duc99nd@gmail.com]**.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If the vulnerability is accepted, we will fix it as soon as possible and credit you for the discovery.
